# BIKETRAILS

Sample collection of trails to ride in Brasov

## Features

- Show trail colors based on difficulty
- Filter trails by difficulty

## Not implemented

X Upload trail
X Delete trail

## Prerequisites

    1. NodeJS
    2. MongoDB
    3. Bootstrap

## Installation

    1. Clone repository
    2. `npm install`
    3. `npm start`
    4. Open <http://localhost:8080/>

## Enviroment variables

MONGODB_URL
